# Output Catalog

This catalog lists the major artifacts generated by the local platform. All outputs are based on
synthetic data and local processing.

| Output | Purpose | Producing command | Downstream use |
| --- | --- | --- | --- |
| `data/raw/identity_events.jsonl` | Synthetic identity lifecycle and role events | `generate-telemetry` | Ingestion, identity governance |
| `data/raw/login_events.jsonl` | Synthetic sign-in activity | `generate-telemetry` | Detection, identity governance |
| `data/raw/endpoint_events.jsonl` | Synthetic endpoint activity | `generate-telemetry` | Detection, risk scoring |
| `data/raw/application_events.jsonl` | Synthetic SaaS/application events | `generate-telemetry` | Detection, governance |
| `data/raw/security_alerts.jsonl` | Synthetic security alerts | `generate-telemetry` | Detection and risk context |
| `data/raw/cloud_activity.jsonl` | Synthetic cloud control-plane activity | `generate-telemetry` | Detection, governance |
| `data/processed/identity_events.csv` | Processed identity telemetry | `ingest-telemetry` | Validation, governance, enrichment |
| `data/processed/login_events.csv` | Processed login telemetry | `ingest-telemetry` | Validation, detections, governance |
| `data/processed/endpoint_events.csv` | Processed endpoint telemetry | `ingest-telemetry` | Validation and detections |
| `data/processed/application_events.csv` | Processed application telemetry | `ingest-telemetry` | Validation and detections |
| `data/processed/security_alerts.csv` | Processed alert telemetry | `ingest-telemetry` | Validation, detections, risk |
| `data/processed/cloud_activity.csv` | Processed cloud telemetry | `ingest-telemetry` | Validation, detections, governance |
| `outputs/ingestion_summary.json` | Dataset ingestion summary | `ingest-telemetry` | Monitoring, dashboards |
| `outputs/data_quality_summary.json` | Validation results and quality score | `validate-telemetry` | Risk, monitoring, reporting |
| `outputs/security_findings.json` | Threat detection findings | `run-detections` | Risk, monitoring, copilot, dashboards |
| `outputs/identity_governance_findings.json` | Identity governance findings | `run-identity-checks` | Risk, monitoring, copilot, dashboards |
| `outputs/identity_review.csv` | Flattened access review table | `run-identity-checks` | Governance review and dashboards |
| `outputs/risk_scores.json` | Entity risk records and risk bands | `score-risk` | Monitoring, copilot, dashboards |
| `outputs/risk_scores.csv` | Flattened risk score table | `score-risk` | Dashboard and spreadsheet analysis |
| `outputs/operational_health_summary.json` | Pipeline health and posture summary | `monitor-platform` | Copilot and executive reporting |
| `outputs/evidence_manifest.json` | Evidence artifact inventory | `monitor-platform` | Compliance evidence and copilot context |
| `outputs/copilot_context.json` | Structured investigation context | `generate-copilot-briefs` | Local copilot reports and dashboards |
| `reports/*.md` | Human-readable evidence and investigation reports | Layer-specific commands | Audit, SOC, governance, and executive review |
| `dashboards/exports/*.csv` | Dashboard-ready curated datasets | `export-dashboard-data` | Power BI, Streamlit, spreadsheet review |
| `dashboards/dashboard_summary.json` | Dashboard export summary and metrics | `export-dashboard-data` | Reporting inventory and portfolio review |
